# しままるタイピング

シマエナガをテーマにしたブラウザ用タイピングゲームのMVPです。

## 遊び方

`index.html` をブラウザで開きます。

出題データは `typing-content.json` から読み込みます。単語、長文、記号・数字、苦手キー、ストーリー、チュートリアル手順、週間プランをJSONで管理できます。ファイルの読み込みに失敗した場合は、`game.js` の既定データで動作します。

- れんしゅう: 15問で終了します。
- ながぶん: 3つの文章を打ち切ると終了します。
- チャレンジ: 60秒でスコアを競います。
- ながぶんチャレンジ: 90秒で長文スコアを競います。
- チュートリアル: ホームポジションと指ごとの担当キーを練習します。
- ストーリー: 5つのステージを進めて星評価を集めます。
- 今日のメニュー: 曜日ごとの練習予定をすぐ始めます。
- 毎日のWPMグラフ: 直近14日のWPM推移を表示し、最高/平均を切り替えられます。
- リマインド: 時刻を変えると自動保存され、オンにすると今日の練習を知らせます。
- BGM: 好きな音声ファイルをドラッグ&ドロップして、無限ループ再生できます。
- 苦手キー: ミスが多いキーを含む練習を自動で出題します。
- 記号・数字: 数字、句読点、記号を直接入力で練習します。
- 話題でながぶん: 貼り付けた文章を分割して長文練習に使えます。
- 難易度: やさしい、ふつう、むずかしい。
- `tsu / tu`、`shi / si`、`chi / ti`、`fu / hu`、`nn / n` など複数のローマ字入力に対応しています。

正しいキーを打つとシマエナガが進み、単語を打ち切ると木の実を集めます。ミスをするとコンボがリセットされます。
